Guia Rápido de Commits Padrões

🔹 O que é o Conventional Commits?
É um padrão de mensagens de commit que segue uma estrutura fixa...

✨ feat

Adiciona uma nova funcionalidade ao projeto.

feat(auth): adiciona autenticação via Google

🚗 fix

Corrige um bug no código.

fix(carrinho): corrige erro ao remover item duplicado

📝 docs

Faz alterações na documentação.

docs: atualiza documentação de instalação

💅 style

Altera a formatação de código, sem afetar a lógica.

style: corrige espaçamento e identação

♻️ refactor

mudança no código que não altera funcionalidade nem corrige bug.

refactor: simplifica a função de cálculo de impostos

🧪 test

Adiciona, altera ou corrige testes existentes.

test: adiciona testes para a função de busca

⚙️ chore

mudanças de manutenção (dependências, configs)

chore: atualiza dependências do npm
